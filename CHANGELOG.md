# Changelog

## [1.0.0] - 2026-03-02

### Added
- **Onboarding**: 4-page setup wizard (name, body stats, activity, goal)
- **Dashboard**: Calorie ring, macro progress bars, water tracker with quick-add
- **Food Database**: 65 Indian & international foods with full nutrition data
- **Add Food**: Search with meal type selector and quantity adjuster
- **Barcode Scanner**: Scan product barcodes via Open Food Facts API
- **Recipe Builder**: Combine ingredients, set servings, auto-calculate nutrition
- **AI Food Search**: Smart search across local DB + Open Food Facts
- **Diary**: Daily meal log grouped by type, swipe-to-delete
- **Meal Planner**: 7-day weekly plan with per-day macro summary
- **Grocery List**: Auto-generated from meal plan with checkable items
- **Insights**: Macro pie chart, weekly calorie bar chart, smart tips
- **Micro-Nutrients**: Track 10 vitamins & minerals with progress indicators
- **Challenges**: 6 built-in health challenges with progress tracking
- **PDF Reports**: Weekly progress report export/print
- **Data Export**: Daily summary text + 30-day JSON export
- **Auth**: Login/signup screens (local, Firebase-ready)
- **Profile**: Personal info, daily targets, BMR/TDEE auto-calculation
- **Settings**: Premium tier placeholder, feature hub, data management
- **5-tab navigation**: Home, Diary, Planner, Insights, Profile
